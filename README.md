# Персональний помічник

Термінальний бот для управління контактами та нотатками з підтримкою пошуку, сортування та збереження даних між сесіями.

## Вимоги

- Python 3.10+

## Запуск

**Mac / Linux:**
```bash
./run.sh
```

**Windows:**
```bash
run.bat
```

**Або напряму:**
```bash
python3 address_book.py
```

## Команди

### Контакти

| Команда | Синтаксис | Опис |
|---|---|---|
| `hello` | `hello` | Привітання |
| `help` | `help` | Список усіх команд |
| `add` | `add <ім'я> <телефон>` | Додати контакт |
| `change` | `change <ім'я> <старий тел> <новий тел>` | Змінити номер телефону |
| `phone` | `phone <ім'я>` | Показати телефони контакту |
| `all` | `all` | Показати всі контакти |
| `add-birthday` | `add-birthday <ім'я> <дата>` | Додати день народження (DD.MM.YYYY) |
| `show-birthday` | `show-birthday <ім'я>` | Показати день народження |
| `birthdays` | `birthdays <кількість днів>` | Контакти з днем народження протягом N днів |
| `add-email` | `add-email <ім'я> <email>` | Додати email |
| `delete` | `delete <ім'я>` | Видалити контакт |
| `find` | `find <запит>` | Пошук контактів за іменем або телефоном |
| `close` / `exit` | `close` | Вийти з програми |

### Нотатки

| Команда | Синтаксис | Опис |
|---|---|---|
| `note-add` | `note-add <заголовок> <текст>` | Додати нотатку |
| `note-search` | `note-search [запит]` | Пошук нотаток за текстом; без запиту — всі |
| `note-edit` | `note-edit <id> <новий текст>` | Змінити вміст нотатки |
| `note-delete` | `note-delete <id>` | Видалити нотатку за id |

### Теги

Теги додаються прямо в текст нотатки через символ `#`:

```
note-add Shopping купити #молоко #хліб #яйця
```

Пошук за тегом:
```
note-search #молоко
```

## Структура проєкту

```
├── address_book.py          # Головний файл, точка входу
├── fields/
│   ├── base_field.py        # Базовий клас поля
│   ├── name.py              # Поле імені
│   ├── phone.py             # Поле телефону з валідацією
│   ├── birthday.py          # Поле дня народження
│   ├── email.py             # Поле email
│   └── email_validator.py   # Валідація email
├── birthdays/
│   ├── get_upcoming_birthdays.py  # Логіка пошуку днів народження
│   └── Utils.py             # Утиліти для дат
├── notebook/
│   ├── models.py            # Модель нотатки (Note)
│   ├── notebook.py          # Клас Notebook (CRUD)
│   ├── handlers.py          # Обробники команд нотаток
│   ├── search.py            # Пошук і сортування нотаток
│   └── storage.py           # Збереження/завантаження нотаток
├── prompt_helper.py         # Підказки команд
├── run.sh                   # Скрипт запуску для Mac/Linux
└── run.bat                  # Скрипт запуску для Windows
```

## Збереження даних

Дані зберігаються автоматично у папці користувача:
- `addressbook.pkl` — контакти
- `notebook.pkl` — нотатки

Дані не втрачаються між перезапусками програми.
