# Консольний бот-помічник (адресна книга та нотатки)

Консольний застосунок для зберігання контактів (ім'я, адреса, телефони, email, день народження) та текстових нотаток з тегами.

## Запуск

### Віртуальне оточення

```bash
python -m venv .venv
```

Windows:
```bash
.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

Linux/macOS:
```bash
source .venv/bin/activate
pip install -r requirements.txt
python main.py
```

### Скрипти

- **Windows:** `run.bat` — активує .venv і запускає `main.py`
- **Linux/macOS:** `./run.sh` — те саме

## Головне меню

При запуску з’являється меню:

- **1** — Адресна книга (команди для контактів)
- **2** — Нотатки (команди для нотаток)
- **0** — Вихід (збереження даних)

У режимі «Адресна книга» або «Нотатки» команда **back** повертає в головне меню.

## Команди

### Контакти (режим «Адресна книга»)

| Команда | Опис |
|--------|------|
| `hello` | Привітання |
| `help [команда]` | Довідка: усі команди або деталі по одній |
| `add <ім'я> <телефон>` | Додати контакт або телефон (телефон — 10 цифр) |
| `change <ім'я> <старий_телефон> <новий_телефон>` | Змінити номер телефону |
| `phone <ім'я>` | Показати телефони контакту |
| `all` | Показати всі контакти (таблиця) |
| `add-birthday <ім'я> <DD.MM.YYYY>` | Додати день народження |
| `show-birthday <ім'я>` | Показати день народження |
| `birthdays [днів]` | Дні народження на найближчі дні (за замовч. 7) |
| `add-email <ім'я> <email>` | Додати email |
| `add-address <ім'я> <адреса>` | Додати або змінити адресу |
| `show <ім'я>` | Повна інформація про контакт |
| `search <запит>` | Пошук за іменем, телефоном, email або адресою |
| `delete <ім'я>` | Видалити контакт |
| `back` | Повернутися в головне меню |

### Нотатки (режим «Нотатки»)

| Команда | Опис |
|--------|------|
| `note-add <заголовок> <текст>` | Додати нотатку (теги через #тег) |
| `note-search [запит]` | Пошук нотаток; без запиту — всі (таблиця) |
| `note-delete <id>` | Видалити нотатку за id |
| `note-edit <id> <новий текст>` | Змінити вміст нотатки |
| `note-sort <tag\|date>` | Сортувати нотатки за тегами або датою |
| `back` | Повернутися в головне меню |

## Валідація

- **Телефон:** рівно 10 цифр
- **Email:** коректний формат
- **День народження:** формат DD.MM.YYYY

## Зберігання даних

Дані зберігаються при виході (0 у головному меню). За замовчуванням:

- Контакти: `addressbook.pkl` або `addressbook.json` (за налаштуванням)
- Нотатки: `notebook.pkl` або `notebook.json`

Файли створюються в каталозі запуску.

## Структура проєкту

```
├── main.py              # Точка входу, головне меню
├── address_book.py      # Історична реалізація (не використовується в main)
├── contacts/            # Адресна книга
│   ├── models.py        # Record, AddressBook
│   ├── handlers.py     # Обробники команд контактів
│   └── storage.py      # Збереження/завантаження контактів
├── notebook/            # Нотатки
│   ├── models.py       # Note
│   ├── notebook.py     # Notebook
│   ├── handlers.py     # Обробники команд нотаток
│   └── storage.py      # Збереження/завантаження нотаток
├── fields/              # Валідовані поля (Name, Phone, Email, Birthday)
├── birthdays/           # Найближчі дні народження
├── core/                # Парсер, декоратори, довідка, таблиці
└── requirements.txt
```
